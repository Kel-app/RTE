## Getting Started

This is the NPM package for Kel's rich text editor (RTE).

There isn't much involved setup, just install the package and import it into your project.

```bash
npm install @kel-app/rte
```

```tsx
import { RTE } from "@kel-app/rte";

const App = () => {
  return <RTE />;
};
```
